# About
A simple blog made in React and Typescript

# Features
Guest
- Read Post
- Post Comments

Admin
- Create New Post
- Edit existing posts
- Delete Comments

# Installation
1. Clone the repo
    ```
    https://github.com/Remiferiaa/blog-api.git
    cd blog-api/client
    ```
2. Run npm install
3. Create a .env.local file with the following field
    ```
    REACT_APP_URL = Your backend api url here
    ```
